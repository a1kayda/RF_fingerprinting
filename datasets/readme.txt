В папке 'raw_preambules' находятся сырые записанные преамбулы для всех 5 устройств. Цифра в конце названия файла соответствует позиции замера (1 или 2).

В папке ‘equalized_preambules’ находятся преамбулы после эквализации.

Файлы ‘autoencode_ch1.mat’ и ‘autoencode_ch2.mat’ содержат датасет для 5 устройств для позиции замера 1 и 2 соответсвенно. В качестве признаков используются выход скрытого слоя автоэнкодера размером 12.

В файлах ‘cpsd_ch1.mat’ и ‘cpsd_ch2.mat’ находиться датасеты, в которых в качестве признаков используется взаимная спектральная плотность мощности. Количество признаков равно 64.

Размеры всех датасетов равны 8782.

Пример загрузки датасета в Python (скрипт read.py):

import scipy.io
 
dict = scipy.io.loadmat("cpsd_ch1.mat")
X = dict['X'] # Признаки
Y = dict['Y'] # Индекс устройства
